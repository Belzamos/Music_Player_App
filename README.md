
#Audio Player App
A simple, intuitive audio player for your favorite music and podcasts.

Features

Play, pause, skip, and shuffle your audio files.
Playlist support: Create and manage custom playlists.
Equalizer: Adjust audio settings for the best listening experience.
Cross-platform: Works on Windows, macOS, and Linux.
Lightweight and fast: Minimal resource usage.

Installation

Clone the repository:

git clone (https://github.com/Belzamos/Music_Player_App.git

Usage

Open the app.
Add your audio files or folders to the library.
Create playlists or play directly from your library.
Use the controls to play, pause, skip, or shuffle tracks.
